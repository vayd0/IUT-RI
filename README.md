# SAE 302 - IUT de Lens

## Description
Projet de développement web dans le cadre de la SAE 302 à l'IUT de Lens.

``[Visualisation](https://sae302.hkth.dev)```

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
- GSAP (animations)
- AOS (Animate On Scroll)
- Tailwind CSS

## Structure du projet
```
├── index.html          # Page principale
├── README.md           # Documentation
├── assets/             # Images et ressources
└── public/             # Fichiers CSS et JS
    ├── style.css       # Styles personnalisés
    └── animations.js   # Animations GSAP
```

## Installation
1. Cloner le dépôt
2. Ouvrir `index.html` dans un navigateur

## Utilisation
Le projet présente une page web interactive avec des animations au scroll et des effets visuels.
